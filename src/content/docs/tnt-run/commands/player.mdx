Player Commands are available to **regular players** and do not require administrative permissions. These commands allow players to interact with **TNT Run** gameplay by joining arenas, leaving matches, and viewing statistics.

All commands can be used via `/tntrun` or its alias `/tr`.

---

## Join Command

Allows a player to join a **TNT Run** arena. The plugin automatically checks arena availability, game state, and capacity before allowing the player to join.

```bash
/tntrun join <arena>
```

---

## Leave Command

Safely removes the player from the current **TNT Run** game. This command can be used at any time during gameplay and will return the player to the configured end location.

```bash
/tntrun leave
```

---

## Stats Command

Displays detailed **TNT Run** statistics through an in-game menu.

By default, the command opens your own statistics, including games played, wins, losses, streaks, and arena records.
When a player name is provided, the same statistics are shown for the specified player.
Statistics can be viewed for both online and offline players.

```bash
/tntrun stats [player]
```
