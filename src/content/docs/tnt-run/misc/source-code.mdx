## Build Notes

The project uses Gradle and builds a shaded plugin JAR.

```bash
./gradlew shadowJar
```

The output JAR is named like:

```text
tnt-run-<version>.jar
```

---

## Main Dependencies

The plugin uses:

* Paper API
* PlaceholderAPI
* Despical Commons
* Command Framework
* Inventory Framework
* File Items
* bStats

Core library packages are relocated in the shaded output to avoid dependency conflicts on production servers.
