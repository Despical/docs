## TNT Run

**TNT Run** is a competitive Minecraft minigame where players run across layers of removable blocks while the floor disappears under their feet.
The goal is simple: keep moving, use your double jumps carefully, avoid falling through every layer, and be the last player alive.

Built for server owners who want a polished arena game, **TNT Run** includes setup menus, join signs, scoreboards, boss bars, spectator menus, per-arena records, statistics, PlaceholderAPI placeholders, and a developer API for custom integrations.
