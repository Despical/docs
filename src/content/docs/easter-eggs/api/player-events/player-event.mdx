Player events are triggered when a user directly interacts with the Easter Eggs system. All events in this category inherit from the base `EasterEggsEvent` class, which provides immediate access to the internal `User` object.

<Aside type="note">
    As the plugin expands, more player-related events will be documented here. Currently, the primary interaction covered is discovering eggs.
</Aside>

---

## FoundNewEggEvent

This event is fired whenever a player interacts with an Easter Egg in the world, regardless of whether it is their first time finding it or they have collected it previously.

### Technical Overview
- **Parent Class:** `EasterEggsEvent`

### Methods

| Method             | Return Type | Description                                                                              |
|:-------------------|:------------|:-----------------------------------------------------------------------------------------|
| `getUser()`        | `User`      | Inherited from `EasterEggsEvent`. Returns the plugin-specific `User` object.             |
| `getEggId()`       | `String`    | Returns the specific ID of the Easter Egg the player interacted with.                    |
| `isAlreadyFound()` | `boolean`   | Returns `true` if the player had already collected this specific egg prior to the event. |

---

## Usage in Listeners

When listening to `FoundNewEggEvent`, you can easily access the `User` object to check player statistics or run custom logic based on whether the egg is new to them.

```java
@EventHandler
public void onEggFound(FoundNewEggEvent event) {
    // Accessing the Easter Eggs User
    User user = event.getUser();
    Player bukkitPlayer = user.getPlayer(); // If you need the standard Bukkit Player

    // Getting the specific egg ID
    String eggId = event.getEggId();

    // Check if the player is finding it for the first time
    if (event.isAlreadyFound()) {
        bukkitPlayer.sendMessage("You have already found the " + eggId + " egg!");
        return;
    }

    // Custom logic for finding a new egg
    bukkitPlayer.sendMessage("Awesome! You discovered a new Easter Egg!");
}
```
