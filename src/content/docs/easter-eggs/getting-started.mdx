This guide provides a quick and easy walkthrough of the system requirements and step-by-step installation process needed
to get the **Easter Eggs** plugin running on your Minecraft server.

## Requirements

Before installing, ensure your server meets the following requirements:

* **Server Software:** **Spigot**, **Paper**, or any compatible fork (e.g., **Purpur**).
* **Minecraft Version:** 1.8 - 1.21+
* **Java Version:** Java 21 or newer.

<Aside type="tip">
    For the best performance and experience, we recommend using [**Paper**](https://papermc.io/downloads/) or its forks.
</Aside>

## Installation

Follow these simple steps to install the plugin:

<Steps>

1.  **Download the Plugin**

    Visit our [Resource Page](https://www.spigotmc.org/resources/116086/) to download the latest version.

2.  **Upload to Server**

    Stop your server and upload the `.jar` file into your server's `/plugins` folder.

3.  **Restart Server**

    Start your server. The plugin will automatically generate the default configuration files.

    ```bash
    [INFO]: [EasterEggs] Loading EasterEggs vX.X.X
    [INFO]: [EasterEggs] Enabling EasterEggs vX.X.X
    [INFO]: [EasterEggs] Successfully loaded X Easter Eggs in XXXms.
    [INFO]: [EasterEggs] Initialization finished. Took XXXms to load.
    [INFO]: [EasterEggs] Join our Discord server: https://discord.gg/uXVU8jmtpU
    ```
</Steps>
